# Constitutions for working on this repository

You must align ALL of these rules below.

## Technology Stack

- TypeScript for all code.
- Zod for input validation.
- Drizzle ORM for database access if we need databases.
- React and Shadcn UI for UI components.
- React Router Framework Mode for routing and application structure.
- Tailwind CSS and CSS Modules for styling.
- Cloudflare Workers for serverless deployment if needed.

## Coding Rules

**THIS IS MANDATORY FOR EVERY WORK**: When you plan, implement and review codes, follow the `domain-driven-development`, `react-router-framework-mode`, `react-router-route-module-patterns`, and `ui-components` skills.
